## Manual
마우스 좌 클릭 총공격<br>
마우스 우 클릭 폭탄<br>
↑방향키 : 앞쪽 이동<br>
↓방향키 : 뒤쪽 이동<br>
→방향키 : 오른쪽 이동<br>
←방향키 : 왼쪽 이동<br>
space : 점프<br>
esc : 옵션창 활성화<br>
